# Курсовая работа 2. Поиск вакансий
## Задание
Напишите программу, которая будет получать информацию о вакансиях с платформы hh.ru в России, 
сохранять ее в файл и позволять удобно работать с ней: добавлять, фильтровать, удалять.

## Взаимодействие с API
Создан абстрактный класс для работы с API сервиса с вакансиями. Реализован класс, 
наследующийся от абстрактного класса, для работы с платформой hh.ru. 
Класс умеет подключаться к API и получать вакансии.

## Работа с вакансиями
Создан класс для работы с вакансиями. В этом классе самостоятельно определены атрибуты, 
такие как название вакансии, ссылка на вакансию, зарплата, 
краткое описание или требования и т. п. (всего не менее четырех атрибутов). 
Класс поддерживает методы сравнения вакансий между собой по зарплате и проводит валидацию данных, 
которыми инициализируются его атрибуты.

## Работа с файлами
Определён абстрактный класс, который обязывает реализовать методы для добавления вакансий в файл, 
получения данных из файла по указанным критериям и удаления информации о вакансиях. 
Создан класс для сохранения информации о вакансиях в JSON-файл

## Вспомогательные функции
Созданы функцию для взаимодействия с пользователем. Функции взаимодействуют с пользователем через консоль. 
Возможности функций следующие:
+ ввести поисковый запрос для запроса вакансий из hh.ru;
+ получить топ N вакансий по зарплате (N запрашивать у пользователя);
+ получить вакансии с ключевым словом в описании.

## Интерфейс управления
+ Создана функция взаимодействия с пользователем.
+ Функция использует экземпляры классов и их методы, реализованные ранее.
+ Не дублируется функциональность, реализованная в классах.
+ Интерфейс взаимодействия с пользователем реализован понятным текстом, т. е. в выводе не используются коллекции, 
только человекочитаемые строки.
+ Функция взаимодействия с пользователем не перегружена.

Тесты написаны для всех функций проекта и находятся в директории `tests/`.

